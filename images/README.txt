IMAGES FOLDER - Oddono's Gelato Website
========================================

This folder is for storing real Oddono's images from Instagram (@oddonos_gelati)

HOW TO USE:
-----------
1. Visit: https://www.instagram.com/oddonos_gelati/
2. Download images you want to use
3. Save them here with descriptive names
4. Update HTML files to reference these images

IMAGE SPECIFICATIONS:
--------------------
- Format: JPG or PNG
- Hero images: 1600x900px (16:9)
- Product images: 800x600px (4:3)
- Max file size: 200KB (compress if needed)

COMPRESS IMAGES AT:
------------------
- https://tinypng.com/
